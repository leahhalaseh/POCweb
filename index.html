Project Features

Our project is called the LoneWorker PoC. This proof of concept aims to promote the safety of lone workers across the UK. We created a windows application that works with the microsoft band to show the potential of this project and how this can be used to help companies with monitoring the safety of their workers.

In this website, you can find a video of the working demonstration of this application as well the requirements it covers in the overview tab. You can also see the extensive research done and hence different approaches attempted for this project under the Research tab. Under the Development tab is where you can find the actual building of this project: there is a Prototype section showing different iterations of the prototype done, there is also a Technical Design tab which gives an idea of how the back end works; The Testing and Evaluation tab shows the user tests done, and finally the Respositories and Further Developments tab was created for other developers to continue with the project.The Management displays all the project and team management with the project including the meetings held, the biweekly reports, and the team management.

We also implemented a Feedback page which acts as a community forum for other users, developers, or any type of stakeholder to give us feedback on the project or leave a comment!
